# Работа со строками
## 3 способа (через массив, через методы String, через RegEx (регулярные выражжения))
1. Напишите метод, который проверяет, является ли строка палиндромом (читается одинаково в обе стороны), игнорируя регистр.
2. Напишите метод, который удаляет из строки все повторяющиеся символы, оставляя только первое вхождение.
3. Напишите метод, который находит первый символ в строке, который не повторяется.
4. Замените все вхождения буквы `'a'` на `'@'` в строке.
5. Подсчитайте количество слов в предложении. Слова разделены одним или несколькими пробелами.
6. Проверьте, равны ли две строки, не обращая внимания на регистр.
7. Проверьте, соответствует ли строка формату российского номера телефона, например:
   =`+7 (999) 123-45-67` или `89991234567`.
8. Найдите и выведите все числа (целые и с плавающей точкой) из текста.
9. Замените все последовательности из двух и более пробелов одним пробелом.
10. Напишите метод, использующий regex для строгой валидации email.
11. Напишите метод, который принимает имя (String) и цену (double) и возвращает строку "Товар: [Имя], Цена: [Цена] руб." с ценой, отформатированной до 2 знаков после запятой.
12. Дан массив строк. Объедините их в одну строку, используя в качестве разделителя "; ".
